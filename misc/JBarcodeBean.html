Bean Properties

In addition to the standard javax.swing.JComponent properties, jbarcodebean.JBarcodeBean has the following properties:

  • code : String - the text to encode
  • narrowestBarWidth : int - the horizontal scaling factor (pixel multiples)
  • checkDigit : boolean - whether a check digit is included in the encoded symbol
  • codeType : BarcodeStrategy - the barcode format
  • barcodeBackground : Color - the color of the spaces between the bars
  • barcodeHeight : int - the height, in pixels, of the barcode
  • showText : boolean - whether to show the encoded text
  • angleDegrees : double - orientation

Bean Methods

  • gifEncode - encode barcode as GIF OutputStream

Bean Events

jbarcodebean.JBarcodeBean does not raise any events other than those of javax.swing.JComponent.
